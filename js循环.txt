js循环

什么是循环？
循环就是在满足条件下，反复执行的一段代码。

for循环：
	for(初始值；条件；值变化）{
		满足条件时执行的代码
	}
	例：for(var i=0;i<10;i++){
		console.log(i)	//依次输出0-9
	}

while 循环：
	while(条件）{
	满足条件执行的代码
	}
	例：
	var i=0;		//为避免死循环，我们也要像for循环一样，给循环一个可变换的用于判断的值
	while(i<10){
		i++;	//对值进行变化
		console.log(“执行了一次”)
	}

do...while循环：
	do{
		执行的代码
	}while(条件）
	该循环语句比较特殊，其会先执行一次循环，在判断是否执行下一次循环。
	如上所示，i等于10，它不管条件是否成立，都会去执行一次，从而打印的i为11

跳出循环：
	break；	//立刻跳出整个循环
	concinue;	//立刻跳出本次循环，开始下一次循环
